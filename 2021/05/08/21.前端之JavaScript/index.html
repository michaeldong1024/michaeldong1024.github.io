JavaScriptжҰӮиҝ°

ECMAScriptе’ҢJavaScriptзҡ„е…ізі»

1996е№ҙ11жңҲпјҢJavaScriptзҡ„еҲӣйҖ иҖ…вҖ“Netscapeе…¬еҸёпјҢеҶіе®ҡе°ҶJavaScriptжҸҗдәӨз»ҷеӣҪйҷ…ж ҮеҮҶеҢ–з»„з»ҮECMAпјҢеёҢжңӣиҝҷй—ЁиҜӯиЁҖиғҪеӨҹжҲҗдёәеӣҪйҷ…ж ҮеҮҶгҖӮж¬Ўе№ҙпјҢECMAеҸ‘еёғ262еҸ·ж ҮеҮҶж–Үд»¶пјҲECMA-262пјүзҡ„з¬¬дёҖзүҲпјҢи§„е®ҡдәҶжөҸи§ҲеҷЁи„ҡжң¬иҜӯиЁҖзҡ„ж ҮеҮҶпјҢе№¶е°Ҷиҝҷз§ҚиҜӯиЁҖз§°дёәECMAScriptпјҢиҝҷдёӘзүҲжң¬е°ұжҳҜ1.0зүҲгҖӮ

иҜҘж ҮеҮҶдёҖејҖе§Ӣе°ұжҳҜй’ҲеҜ№JavaScriptиҜӯиЁҖеҲ¶е®ҡзҡ„пјҢдҪҶжҳҜжІЎжңүз§°е…¶дёәJavaScriptпјҢжңүдёӨдёӘж–№йқўзҡ„еҺҹеӣ гҖӮдёҖжҳҜе•Ҷж ҮпјҢJavaScriptжң¬иә«е·Іиў«NetscapeжіЁеҶҢдёәе•Ҷж ҮгҖӮиҖҢжҳҜжғідҪ“зҺ°иҝҷй—ЁиҜӯиЁҖзҡ„еҲ¶е®ҡиҖ…жҳҜECMAпјҢиҖҢдёҚжҳҜNetscapeпјҢиҝҷж ·жңүеҲ©дәҺдҝқиҜҒиҝҷй—ЁиҜӯиЁҖзҡ„ејҖеҸ‘жҖ§е’Ңдёӯз«ӢжҖ§гҖӮ

еӣ жӯӨECMAScriptе’ҢJavaScriptзҡ„е…ізі»жҳҜпјҢеүҚиҖ…жҳҜеҗҺиҖ…зҡ„и§„ж јпјҢеҗҺиҖ…жҳҜеүҚиҖ…зҡ„дёҖз§Қе®һзҺ°гҖӮ

ECMAScriptзҡ„еҺҶеҸІ

е№ҙд»Ҫ еҗҚз§° жҸҸиҝ°
1997 ECMAScript 1 з¬¬дёҖдёӘзүҲжң¬
1998 ECMAScript 2 зүҲжң¬еҸҳжӣҙ
1999 ECMAScript 3 ж·»еҠ жӯЈеҲҷиЎЁиҫҫејҸж·»еҠ try/catch
ECMAScript 4 жІЎжңүеҸ‘еёғ
2009 ECMAScript 5 ж·»еҠ вҖқstrict modeвҖқдёҘж јжЁЎејҸж·»еҠ JSONж”ҜжҢҒ
2011 ECMAScript 5.1 зүҲжң¬еҸҳжӣҙ
2015 ECMAScript 6 ж·»еҠ зұ»е’ҢжЁЎеқ—
2016 ECMAScript 7 еўһеҠ жҢҮж•°иҝҗз®—з¬ҰпјҲ**пјүеўһеҠ Array.prototype.includes

жіЁпјҡES6е°ұжҳҜжҢҮECMAScript 6гҖӮ

е°Ҫз®Ў ECMAScript жҳҜдёҖдёӘйҮҚиҰҒзҡ„ж ҮеҮҶпјҢдҪҶе®ғе№¶дёҚжҳҜ JavaScript е”ҜдёҖзҡ„йғЁеҲҶпјҢеҪ“з„¶пјҢд№ҹдёҚжҳҜе”ҜдёҖиў«ж ҮеҮҶеҢ–зҡ„йғЁеҲҶгҖӮе®һйҷ…дёҠпјҢдёҖдёӘе®Ңж•ҙзҡ„ JavaScript е®һзҺ°жҳҜз”ұд»ҘдёӢ 3 дёӘдёҚеҗҢйғЁеҲҶз»„жҲҗзҡ„пјҡ

  • ж ёеҝғпјҲECMAScriptпјү
  • ж–ҮжЎЈеҜ№иұЎжЁЎеһӢпјҲDOMпјү Document object model (ж•ҙеҗҲjsпјҢcssпјҢhtml)
  • жөҸи§ҲеҷЁеҜ№иұЎжЁЎеһӢпјҲBOMпјү Broswer object modelпјҲж•ҙеҗҲjsе’ҢжөҸи§ҲеҷЁпјү
